📝 Descripción
Este proyecto es el resultado de un desarrollo en equipo que demuestra nuestras capacidades técnicas en la creación de software utilizando **C#** y el motor **Unity**. En él, hemos implementado soluciones de lógica de programación aplicadas a un entorno de tiempo real, enfocándonos en la limpieza del código, la modularidad y la arquitectura del sistema.

## 🚀 Características Técnicas
* **Programación en C#:** Uso de programación orientada a objetos (POO) para el control de personajes y mecánicas de juego.
* **Desarrollo Colaborativo:** Implementación de un flujo de trabajo organizado para la integración de scripts y recursos de diferentes áreas.
* **Gestión de Estados:** Implementación de lógica para el manejo de fases del juego y comportamientos de IA.
* **Optimización:** Estructuración de código para asegurar un rendimiento fluido y eficiente en la ejecución.

## 👥 Colaboradores
Este proyecto fue desarrollado en conjunto con un equipo de ingenieros. Puedes consultar los perfiles individuales de los desarrolladores en la sección de **Contributors** de este repositorio.

## 🛠️ Tecnologías y Herramientas
* **Motor:** Unity.
* **Lenguaje:** C# (.NET).
* **Gráficos y Modelado:** Blender (utilizado para la creación y optimización de assets 3D).
* **Control de Versiones:** Git / GitHub.
